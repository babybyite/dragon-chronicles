# Ravenwood Mystery

Ravenwood Mystery is an Expo/React Native murder mystery prototype set in Ravenwood Manor.

The active work now lives in `apps/mobile` and focuses on the mansion investigation: detectives, residents, secrets, room exploration, journals, maps, relationships, and family trees.

## Run

```bash
cd apps/mobile
pnpm install
pnpm run web
```

## Useful Scripts

```bash
pnpm run typecheck
pnpm run ai
```
